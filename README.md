# Stock_Price_Analysis
# 📈 Stock Price Analysis Dashboard

An interactive **Stock Price Analysis Web App** built using **Python, Streamlit, Plotly, and yFinance**.
This application allows users to analyze stock performance, visualize price trends, and estimate investment returns using real-time market data.

---

## 🚀 Live Application

🔗 https://market-stock-analysis.streamlit.app/

---

## 📊 Features

* 📈 **Real-time Stock Data** – Fetches stock market data using yFinance.
* 📉 **Candlestick Chart Visualization** – Interactive candlestick chart using Plotly.
* 📊 **Moving Average Indicators** – Displays 50-day and 200-day moving averages.
* 📉 **Daily Returns Analysis** – Shows percentage change in stock prices.
* 💰 **Investment Profit/Loss Estimation** – Calculates potential profit based on investment amount.
* 📥 **Download Data** – Export stock data as CSV.
* 📊 **Statistical Summary** – Displays descriptive statistics of stock data.
